# Flag API's
### flag.set_flag
```cpp
flag.set_flag(flag_name : string, flag_value : bool) -> void
```
Sets a flag for function features
### flag.list_flags
```cpp
flag.list_flags() -> {[name : string], [value : any]}
```
List all flags
### Current list of supported flags
```cpp
Feature_Flags = {
    ["isDebug"] = false,
    ["allowDownload"] = true,
    ["allowSystemCommand"] = false,
    ["disableAllAPI"] = false,
}
```
### isDebug
Prints more info when running functions

### allowDownload
Allows downloading from URL's
